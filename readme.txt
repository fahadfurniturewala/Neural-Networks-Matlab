Instructions to run the function neural_network in MATLAB.
The name of the function is neural_network.
In the MATLAB command window enter the following commands.


1.type  " neural_network pendigits_training.txt pendigits_testing.txt '2' '0' '50' " in the command window in Matlab
2.Press enter



The first argument is the training file name which can be changed as and when needed.
The second argument is the testing file name which can be changed as and when needed.
The third argument is the number of layers that make up the neural network.
Th fourth argument is the number of units in the hidden layer if it exists at all.
The fifth argument is the numbr of training rounds that need to be completed.

The other possible running combinatins are:

neural_network pendigits_training.txt pendigits_testing.txt '3' '20' '20'
